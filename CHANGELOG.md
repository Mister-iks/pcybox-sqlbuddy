# Changelog

All notable changes to SQL Buddy will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-XX

### Added
- Initial release of SQL Buddy
- Support for MySQL and PostgreSQL databases
- OpenAI GPT and Anthropic Claude integration
- Automatic database schema extraction
- Natural language to SQL query generation
- Query validation and safety checks
- Query optimization suggestions
- Query explanation feature
- Multiple query variations generation
- CLI interface with rich formatting
- Context manager support
- Comprehensive test suite
- Documentation and examples
